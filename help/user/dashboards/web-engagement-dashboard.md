# Web engagement dashboard

The Web engagement dashboard provides visibility into how web visitors interact with key content. It segments data across account industries and regions to help you understand engagement trends. Use this dashboard to support strategic decision-making by surfacing web behavior patterns that inform content strategy and account targeting.

To access the _Web Engagement dashboard_, select the **[!UICONTROL Dashboard]** item in the left navigation. Then select the **[!UICONTROL Web engagement]** tab at the top of the page.

![Web engagement dashboard](./assets/web-engagement-dashboard.png){width="700" zoomable="yes"}

## Filter the data

Click the _Filter_ ( ![Filter icon](../assets/do-not-localize/icon-filter.svg) ) icon at the top left to filter the displayed data using any of the following attributes:

* **[!UICONTROL Account Region]** - Filters the data by one or more selected geographic regions that are associated with accounts.
* **[!UICONTROL Account Industry]** - Filters the data by one or more selected industry classifications that are  associated with accounts.
* **[!UICONTROL Date Range]** - Filters the data by the selected date range. The default range is the current date.

![Filter the displayed data by attributes](./assets/web-engagement-dashboard-filters-dialog.png){width="500"}

Select as many values for each attribute that you want to use to filter the data and click **[!UICONTROL Apply]**.

## [!UICONTROL Top page views] {#top-page-views}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_top_page_views"
>title="Top page views"
>abstract="The most frequently viewed pages on your website, helping you identify which content drives the most traffic."

This table displays the top 10 most viewed web pages, helping you identify which content resonates most with visitors. The data includes:

| Column | Description |
| ------ | ----------- |
| Page name | The name or title of the web page. |
| Total views | The total number of times that the page was viewed. |
| Known visitors (%) | The percentage of page views that are attributed to known (identified) visitors. |
| Unknown visitors (%) | The percentage of page views that are attributed to unknown (anonymous) visitors. |

![Top page views table](./assets/web-engagement-dashboard-top-page-views.png){width="650" zoomable="yes"}

## [!UICONTROL Page views by account region] {#page-views-by-region}

>[!CONTEXTUALHELP]
>id="ajo-b2b_web_engagement_page_views_by_region"

This visualization displays visitor counts segmented by the account region. It illustrates how web traffic varies across different geographic regions, enabling you to tailor content and campaigns to regional audiences. Hover over a bar in the chart to see the details, including:

* Name of account region
* Count of page views

## Engage with the data

To engage with the data, use the _More_ (**...**) menu at the top-right of each chart and choose **[!UICONTROL View more]** to view extended data and insights.

The displayed pop-up includes a chart and a table that show the breakdown of the data.

To download the data, click **[!UICONTROL Download CSV]** at the top right of the data table.

![Top page views table View more popup - Click Download CSV](./assets/web-engagement-dashboard-top-page-views-view-more.png){width="700" zoomable="yes"}

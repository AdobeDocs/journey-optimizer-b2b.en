# Content authoring - link tracking

The link tracking panel provides a centralized view where you can easily locate each URL in the content. Click the _[!UICONTROL Links]_ icon ( ![Show links icon](../assets/do-not-localize/icon-links.svg) ) on the left to display all the linked URLs in your content. These are the links within text, button, or image components.

To define tracking for a listed content link, click the _Edit_ ( ![Edit icon](../user/assets/do-not-localize/icon-edit.svg) ) icon and modify the **[!UICONTROL Tracking Type]** or **[!UICONTROL Label]**.

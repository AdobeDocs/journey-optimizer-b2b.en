# Email templates

Email templates in [!DNL Adobe Journey Optimizer B2B Prime] allow content specialists to build reusable email layouts independently, so marketing teams can apply them across account journeys without rebuilding designs each time.

>[!NOTE]
>
>In this Beta release, only email templates are supported.

When a template is applied to a new email, the content is copied at that point. Edits to the template do not affect emails that were previously created from it.

## Access email templates {#access}

Navigate to **[!UICONTROL Content Management]** > **[!UICONTROL Templates]** in the left navigation. The template library opens in table format, sorted by most recently modified by default.

Toggle between list view and grid view (thumbnail preview) using the icons at the upper right of the library.

### Search and filter {#search-filter}

Use the search bar to find a template by name. Click any column header to sort the list. Click **[!UICONTROL Filters]** to narrow the list by tags or to show only templates you created.

To customize which columns appear in the table, click the **[!UICONTROL Customize table]** icon and select the columns you want to display.

## Organize templates with folders {#folders}

Create folders to group templates by team, campaign, brand, or any structure that fits your workflow.

**To create a folder:**

1. In the template library, click **[!UICONTROL Create folder]**.
1. Enter a folder name and click **[!UICONTROL Confirm]**.

**To move a template into a folder:**

1. Click the **[!UICONTROL More]** icon ( ... ) next to the template.
1. Select **[!UICONTROL Move to folder]**.
1. Select the destination folder and click **[!UICONTROL Move]**.

## View template details {#view-details}

Click a template name to open its details page. From here you can review properties such as creation date and modification history, and see which journey emails use the template on the **[!UICONTROL Used By]** tab.

## Manage templates {#manage}

### Edit a template

Click a template name to open the details page, then click **[!UICONTROL Edit email template]**. You can also click the **[!UICONTROL More]** icon ( ... ) in the template list and select **[!UICONTROL Edit]**.

### Duplicate a template

Click the **[!UICONTROL More]** icon ( ... ) next to the template and select **[!UICONTROL Duplicate]**. The copy opens so you can rename it and make adjustments.

### Delete a template

Click the **[!UICONTROL More]** icon ( ... ) next to the template and select **[!UICONTROL Delete]**. Confirm the deletion when prompted.

>[!NOTE]
>
>Deleting a template does not affect journey emails that were already created from it. Those emails retain their own independent copy of the content.

### Bulk delete

Select the checkbox next to each template you want to remove, or select the header checkbox to select all visible templates. Click **[!UICONTROL Delete]** in the bulk action bar and confirm.

## Use a template in an email {#use-in-journey}

When authoring an email in a journey, the **[!UICONTROL Select design template]** section lets you choose from saved templates or sample templates. Click a template thumbnail to preview it, then click **[!UICONTROL Use this template]** to apply it. For information about how you can use templates in your email authoring workflow, see [Email content authoring](./email-authoring.md#templates).

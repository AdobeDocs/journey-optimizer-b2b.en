# Home page

The _Home_ page is your launch pad in Adobe Journey Optimizer B2B Prime. Select the _Home_ icon at the top of the left navigation to open it. The page greets you with **Welcome [name], what can we build today?** and is designed to take you from idea to action in one click.

![Home page](./assets/home-page.png){width="800" zoomable="yes"} 

## Input box

A large _Ask a question or start a task_ input sits at the center of the home page. Enter anything here to start working with the AI conversational panel, or use the _Attach_ ( **+**) icon to add a file (such as a campaign brief or a CSV) before you send. This is the same AI Assistant available throughout Marketing Management, but starting here opens a fresh conversation.

## Intent buttons

Under the input box, four quick-start buttons frame what you want to accomplish:

| Button | Usage |
|--------|------------|
| **Plan** | Shape strategy — plan campaigns, programs, and audiences before building. |
| **Create** | Build things — programs, journeys, emails, people lists. |
| **Measure** | Look at performance — reports, engagement, send-time results. |
| **Optimize** | Improve what's live — traffic control, scoring, personalization. |

## Quick access

There are four shortcut cards that you can use to quickly jump directly to a working area:

| Card | What you can access |
|------|-----------------|
| **Marketing** | Programs, journeys, audiences (Marketing Management). |
| **Assets** | Emails, pages, templates, tokens. |
| **Reports** | Performance and engagement metrics. |
| **Destinations** | Downstream systems and syncs. |

## Skills

A row of _Skill_ cards showcases high-value AI workflows you can launch immediately:

- **Create persona-based audiences** — Generate segments grouped by persona signals and firmographic fit, ready to drop into a program.
- **Build omni-channel programs** — Compose programs spanning email, web, and downstream destinations from a single brief.
- **Create hyper-personalized emails** — Draft on-brand emails personalized per recipient with tokens and dynamic content blocks.
- **Fit and intent-based journey traffic control** — Route contacts across journeys based on account fit and live buyer-intent signals.

Click a skill card to start a conversation primed for that workflow. Skills are the same building blocks available inside the chat panel — the cards are a discoverable entry point.

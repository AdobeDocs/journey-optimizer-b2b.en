# Marketing management

Select the _Marketing Management_ icon (megaphone) in the left navigation to open your central workspace for planning, organizing, and activating marketing. From here you can manage [programs](./programs.md), build [journeys](./person-journeys.md), maintain [people lists](../audiences/people-lists.md), and create [content](../content/digital-asset-management.md) — all in one place.

Marketing Management uses a three-region layout: a chat panel on the left, a workspace in the center, and a program tree on the right.

![Marketing management workspace](./assets/marketing-home-page.png){width="800" zoomable="yes"} 

## Chat panel {#chat-panel}

The chat panel runs alongside your work so you can ask the [AI Assistant](../agents/chat-interface.md) to do things in context. The panel header includes the following controls:

| Control | Description |
|---------|-------------|
| **New conversation** | Start a fresh conversation. |
| **Conversation history** | Browse and reopen past conversations. |
| **Swap panels** | Switch the chat panel to the other side. |
| **Collapse** | Hide the panel to maximize workspace space. |

The input reads _Ask anything, or type / for skills…_ When you have an asset selected in the workspace, the input becomes context-aware (for example, _Ask about "[asset name]"…_) so your questions apply directly to what you're viewing.

## Workspace {#workspace}

The center workspace is where assets open for viewing and editing.

When nothing is selected, the workspace shows a welcome state — **Welcome back. Revenue won't generate itself.** — with a prompt to select a folder or asset from the program tree.

- **Recently accessed** — A list of the [journeys](./person-journeys.md), [programs](./programs.md), and folders you touched most recently, each with a relative timestamp. Click any row to reopen it.
- **Asset view** — Once you select something from the program tree, its detail opens in the workspace.
- **Inventory pages** — Selecting a folder or program shows an inventory table of its contents ([programs](./programs.md), [journeys](./person-journeys.md), [people lists](../audiences/people-lists.md), emails, and so on). Inventory tables support column resizing, show/hide columns, sorting, and search. Selecting a people list, for example, opens its members inventory.

## Program tree {#program-tree}

The right panel displays the navigation tree for your marketing-related assets. It provides functional tools that you can use:

- A **Create program** button at the top (see [Programs](./programs.md)).
- A **Search** box to find assets by name.
- A hierarchical folder tree rooted at _Marketing Activities/Default_, containing folders, [programs](./programs.md), and [journeys](./person-journeys.md). Expand folders to drill in; the **…** menu on each row exposes per-asset actions.

Select an item in the program tree to open the details in the center workspace.
